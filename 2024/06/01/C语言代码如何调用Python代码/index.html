背景

本文主要记录C语言如何调用Python代码，Python有很多方便的库，但是C语言可能没有，使用C语言调用Python代码可以让C语言变得更强。

在C语言中调用Python代码主要使用Python C API。

基本原理

在C语言中调用Python代码的原理是通过嵌入Python解释器来实现的。实际上，C程序启动并嵌入了一个Python解释器实例，这使得可以在C程序中执行Python代码。

这样做的好处就是可以直接使用Python代码，坏处是增加了使用的资源。

hello world

#include <stdio.h>
#include <Python.h>

int main() {
    Py_Initialize();
    PyRun_SimpleString("print(\"hello world\")");
    Py_Finalize();
    return 0;
}

编译运行，这里需要链接python的库，我使用的Python3.9版本，需要根据版本修改头文件和链接库的版本

$ gcc hello.c -I /usr/include/python3.9 -lpython3.9 -o hello
$ ./hello
hello world

步骤

  1. 初始化Python解释器：使用Py_Initialize()初始化Python解释器。
  2. 执行Python代码：使用PyRun_SimpleString()等API执行Python代码。
  3. 获取Python函数：从Python模块中获取Python函数，并调用它们。
  4. 传递参数和返回值：在C和Python之间传递参数和返回值。
  5. 关闭Python解释器：使用Py_Finalize()关闭Python解释器。

从文件加载Python代码

只是执行简单Python语句并没有太大作用，还可以从现成的Python脚本文件中，加载并运行代码：

hello.py如下：

import time
import requests
while 1:
    print("hello world")
    time.sleep(5)
    print(requests.get('https://www.baidu.com'))

C代码如下：

#define PY_SSIZE_T_CLEAN
#include <stdio.h>
#include <Python.h>

int main() {
    char file_name[] = "hello.py";

    FILE* fp;

    Py_Initialize();

    fp = _Py_fopen(file_name, "r");
    PyRun_SimpleFile(fp, file_name);

    Py_Finalize();

    return 0;
}

一个更复杂的实例

C语言调用Python的方法实例：

my_module.py

def my_function(x):
    print(f"Received {x} in Python")
    return x + 1

complex.c：

#include <Python.h>
#include <stdio.h>
#include <stdlib.h>

int main() {
    // 初始化Python解释器
    Py_Initialize();

    // 简单执行Python代码
    PyRun_SimpleString("print('Hello from Python!')");
    // 需要导入Python模块路径
    //PyRun_SimpleString("import sys; sys.path.append('./')");

    // 导入Python模块
    PyObject *pName, *pModule, *pFunc;
    PyObject *pArgs, *pValue;

    pName = PyUnicode_DecodeFSDefault("my_module");
    pModule = PyImport_Import(pName);
    Py_DECREF(pName);

    if (pModule != NULL) {
        pFunc = PyObject_GetAttrString(pModule, "my_function");
        if (PyCallable_Check(pFunc)) {
            // 创建参数
            pArgs = PyTuple_New(1);
            pValue = PyLong_FromLong(42);
            if (!pValue) {
                Py_DECREF(pArgs);  
                Py_DECREF(pModule);
                fprintf(stderr, "Cannot convert argument\n");
                return 1;
            }
            PyTuple_SetItem(pArgs, 0, pValue);

            // 调用Python函数
            pValue = PyObject_CallObject(pFunc, pArgs);
            Py_DECREF(pArgs);
            if (pValue != NULL) {
                printf("Result of call: %ld\n", PyLong_AsLong(pValue));
                Py_DECREF(pValue);
            } else {
                Py_DECREF(pFunc);
                Py_DECREF(pModule);
                PyErr_Print();
                fprintf(stderr, "Call failed\n");
                return 1;
            }
        } else {
            if (PyErr_Occurred())
                PyErr_Print();
            fprintf(stderr, "Cannot find function \"my_function\"\n");
        }
        Py_XDECREF(pFunc);
        Py_DECREF(pModule);
    } else {
        PyErr_Print();
        fprintf(stderr, "Failed to load \"my_module\"\n");
        return 1;
    }

    // 关闭Python解释器
    Py_Finalize();
    return 0;
}

需要指定pythonpath或者在代码中添加my_moudle的路径，不然会报错。

报错如下：

ModuleNotFoundError: No module named 'my_module'
Failed to load "my_module"

推荐使用PYTHONPATH

$ export PYTHONPATH=~/C_with_py

Py_DECREF和Py_XDECREF解释

Py_DECREF 和 Py_XDECREF 是用于管理Python对象引用计数的宏。

当你在C代码中使用Python对象时，这些宏可以帮助你正确管理内存，避免内存泄漏。

总结

C语言调用Python代码流程还是比较简单的，但是主要需要手动管理内存和相关对象的引用，不然可能造成内存泄漏，需要小心。
