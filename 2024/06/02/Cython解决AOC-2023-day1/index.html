题目大意

第一部分

在这个问题中，每行文本最初包含一个校准值（calibration value），但现在这些行已经被修改过了。每一行的校准值可以通过将该行中的第一个数字和最后一个数字组合起来形成一个两位数来恢复。

让我们看看一些示例：

  1. 1abc2：这一行的第一个数字是 1，最后一个数字是 2，所以组合成的两位数是 12。
  2. pqr3stu8vwx：这一行的第一个数字是 3，最后一个数字是 8，所以组合成的两位数是 38。
  3. a1b2c3d4e5f：这一行的第一个数字是 1，最后一个数字是 5，所以组合成的两位数是 15。
  4. treb7uchet：这一行的第一个数字是 7，最后一个数字也是 7，所以组合成的两位数是 77。

根据这些示例，我们可以归纳出每行的处理步骤：

  1. 找到该行中的第一个数字。
  2. 找到该行中的最后一个数字。
  3. 将这两个数字组合成一个两位数。

这就是如何从每行文本中恢复校准值的方法。

第二部分

第二部分题目又改了，1-9部分既可以用数字描述，又可以用英文描述了，简单方法就是使用replace做预处理，把英文的部分都给替换为数字，然后再用第一部分都方法去运行程序。

Python版本解决

import time
def find_calibration_values_python(lines):
    all_res = 0
    for line in lines:
        nums = [i for i in line if i in '0123456789']
        if len(nums) == 1:
            nums = nums[0] * 2
        else:
            nums = [nums[0], nums[-1]]
        all_res += 10 * int(nums[0]) + int(nums[1])
    return all_res

all_lines = []
with open('in.txt') as fd:
    for line in fd:
        all_lines.append(line.strip())


s = time.time()
for i in range(100):
    b = find_calibration_values_python(all_lines)
print(time.time() - s)
print(b)

Cython版本解决

%%cython 

import time

from libc.stdlib cimport malloc, free
from libc.stdio cimport printf, fflush, stdout
from libc.string cimport strcpy

cdef int is_number(char c):
    return c >= b'0' and c <= b'9'

def find_single_calibration_values(str s):
    cdef char* buf
    cdef int length
    cdef int i
    length = len(s) + 1
    cdef int array[2]
    array[0] = -1
    array[1] = -1

    cdef int ret
    
    buf = <char*>malloc(sizeof(char) * length)
    if buf == NULL:
        raise MemoryError()

    strcpy(buf, s.encode('utf-8'))
    # 打印 buf 的内容
    i = 0
    while buf[i] != b'\0':
        if is_number(buf[i]): 
            if array[0] == -1:
                # 48是'0'的ASCII码
                array[0] = buf[i] - 48
            elif array[0] != -1:
                array[1] = buf[i] - 48
        i += 1

    
    ret = 0
    if array[1] == -1:
        array[1] = array[0]

    ret = 10 * array[0] + array[1]
    free(buf)
    return ret

cpdef int find_calibration_values(list[str] lines):
    cdef int all_res
    all_res = 0
    for line in lines:
        all_res += find_single_calibration_values(line)
    return all_res


all_lines = []
with open('in.txt') as fd:
    for line in fd:
        all_lines.append(line.strip())


s = time.time()
for i in range(100):
    b = find_calibration_values(all_lines)
print(b)
print(time.time() - s)
