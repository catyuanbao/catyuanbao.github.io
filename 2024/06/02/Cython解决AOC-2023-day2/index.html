背景

本文主要记录解决AOC-2023-day2的问题过程，主要使用Python和Cython分别测试效果。

题目大意

第一部分

为了获取信息，一旦袋子里装满了方块，精灵就会把手伸进袋子里，随便抓一把方块，给你看，然后把它们放回袋子里。他每场比赛都会这样做几次。

你玩了几个游戏，并记录每个游戏的信息（你的谜题输入）。每个游戏都列出了其ID号（如游戏11中的11：…），然后是从包中显示的立方体子集的分号分隔列表（如3个红色、5个绿色、4个蓝色）。

例如，一些游戏的记录可能是这样的：

游戏一：3个蓝色，4个红色；1个红色，2个绿色，6个蓝色；2个绿色

游戏二：1个蓝色，2个绿色；3个绿色，4个蓝色，1个红色；1个绿色，1个蓝色

第三场：8个绿色，6个蓝色，20个红色；5个蓝色，4个红色，13个绿色；5个绿色，1个红色

游戏四：1个绿色，3个红色，6个蓝色；3个绿色，6个红色；3绿色，15蓝色，14红色

第五场：6红，1蓝，3绿；2个蓝色，1个红色，2个绿色

在游戏1中，三组立方体从袋子中露出（然后再放回）。第一组是3个蓝色方块和4个红色方块；第二组是1个红色方块、2个绿色方块和6个蓝色方块；第三组只有2个绿色方块。

精灵首先想知道，如果袋子里只有12个红色方块、13个绿色方块和14个蓝色方块，哪些游戏是可能的？

在上面的例子中，如果袋子已经装载了该配置，那么游戏1、2和5将是可能的。然而，第三场比赛是不可能的，因为有一次精灵一次给你看了20个红色方块；同样，第四场比赛也不可能，因为精灵同时向你展示了15个蓝色方块。如果你把可能的游戏ID加起来，你会得到8个。

如果袋子里只装了12个红色方块、13个绿色方块和14个蓝色方块，那么确定哪些游戏是可能的。这些游戏的ID总和是多少？

这个问题就是有一个约束条件，根据约束条件判断，给定的游戏是否满足要求，不能超过红色12个，绿色13个，蓝色14个。

第二部分

游戏一：3个蓝色，4个红色；1个红色，2个绿色，6个蓝色；2个绿色

游戏二：1个蓝色，2个绿色；3个绿色，4个蓝色，1个红色；1个绿色，1个蓝色

第三场：8个绿色，6个蓝色，20个红色；5个蓝色，4个红色，13个绿色；5个绿色，1个红色

游戏四：1个绿色，3个红色，6个蓝色；3个绿色，6个红色；3绿色，15蓝色，14红色

第五场：6红，1蓝，3绿；2个蓝色，1个红色，2个绿色

在游戏1中，游戏可以使用4个红色、2个绿色和6个蓝色方块。如果任何一种颜色少了一个立方体，游戏就不可能了。

第二场比赛本可以用至少1个红色、3个绿色和4个蓝色方块进行。

第三场比赛必须至少有20个红色方块、13个绿色方块和6个蓝色方块。

第四场比赛需要至少14个红色方块、3个绿色方块和15个蓝色方块。

第五场比赛需要不少于6个红色、3个绿色和2个蓝色方块。

一组立方体的幂等于红色、绿色和蓝色立方体的数量相乘。游戏1中的最小立方体集的幂是48。在第2-5场比赛中，这一数字分别为12、1560、630和36。把这五个幂加起来就得到2286。

对于每个游戏，找到必须存在的最小立方体集。这些集合的幂和是多少？

就是找到每个游戏中每个颜色的最大值，把三个颜色的乘起来，最后所有游戏的结果加和。

Python版本解决

import time

all_lines = []
with open('in2.txt') as fd:
    for line in fd:
        all_lines.append(line.strip())


def single_match(record):
    total = [12,13,14]
    num_str, ball_str = record.split(':')
    num = num_str[4:]
    ball = ball_str.split(';')
    for i in ball:
        for j in i.split(','):
            if 'red' in j and total[0] < int(j[:-4]):
                return 0
            elif 'green' in j and total[1] < int(j[:-6]):
                return 0
            elif 'blue' in j and total[2] < int(j[:-5]):
                return 0
    return int(num)
        
def compute_all_sum(all_lines):
    return sum((single_match(line) for line in all_lines))

s = time.time()
for i in range(1000):
    compute_all_sum(all_lines)
print(time.time() - s)

运行时间：2.8653945922851562

import time

def single_match(record):
    total = [0,0,0]
    num_str, ball_str = record.split(':')
    ball = ball_str.split(';')
    for i in ball:
        for j in i.split(','):
            if 'red' in j:
                red_num = int(j[:-4])
                total[0] = red_num if total[0] < red_num else total[0]

Cython版本解决

%%cython 

import time

all_lines = []
with open('in2.txt') as fd:
    for line in fd:
        all_lines.append(line.strip())

cdef int a_lees_than_b(int a, int b):
    return 1 if a < b else 0


cpdef single_match(record):
    cdef int total[3]
    total[0] = 12
    total[1] = 13
    total[2] = 14
    cdef int num
    
    num_str, ball_str = record.split(':')
    num = int(num_str[4:])
    ball = ball_str.split(';')

    for i in ball:
        for j in i.split(','):
            if 'red' in j and a_lees_than_b(total[0], int(j[:-4])):
                return 0
            elif 'green' in j and a_lees_than_b(total[1], int(j[:-6])):
                return 0
            elif 'blue' in j and a_lees_than_b(total[2], int(j[:-5])):
                return 0
    return num

def compute_all_sum(all_lines):
    return sum((single_match(line) for line in all_lines))

s = time.time()
for i in range(1000):
    compute_all_sum(all_lines)
print(time.time() - s)

%%cython 

import time

all_lines = []
with open('in2.txt') as fd:
    for line in fd:
        all_lines.append(line.strip())

cdef int a_more_than_b(int a, int b):
    return a if a > b else b


cpdef single_match(record):
    cdef int total[3]
    total[0] = 0
    total[1] = 0
    total[2] = 0
    cdef int num
    cdef int red_num
    cdef int green_num
    cdef int blue_num
    
    num_str, ball_str = record.split(':')
    ball = ball_str.split(';')

    for i in ball:
        for j in i.split(','):
            if 'red' in j:
                red_num = int(j[:-4])
                total[0] = a_more_than_b(total[0], red_num)
            elif 'green' in j:
                green_num = int(j[:-6])
                total[1] = a_more_than_b(total[1], green_num)
            elif 'blue' in j:
                blue_num = int(j[:-5])
                total[2] = a_more_than_b(total[2], blue_num)

    return total[0] * total[1] * total[2]

def compute_all_sum(all_lines):
    return sum((single_match(line) for line in all_lines))

s = time.time()
for i in range(1000):
    compute_all_sum(all_lines)
print(time.time() - s)

运行时间：2.1201305389404297
