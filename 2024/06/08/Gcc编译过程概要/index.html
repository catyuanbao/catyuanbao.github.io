背景

本文主要记录C/C++程序的编译步骤，了解其中做了哪些事情。

编译过程

C/C++程序，从源代码到二进制文件的编译过程如下：

+------------------------+
|    hello.c (源代码)    |
+-----------+------------+
            |
            v
+------------------------+
|    预处理 (Preprocessing)     |
|  (扩展宏定义和包含文件)    |
+-----------+------------+
            |
            v
+------------------------+
|    hello.i (预处理后的代码)    |
+-----------+------------+
            |
            v
+------------------------+
|    编译 (Compilation)        |
|  (转换为汇编代码)      |
+-----------+------------+
            |
            v
+------------------------+
|    hello.s (汇编代码)        |
+-----------+------------+
            |
            v
+------------------------+
|    汇编 (Assembly)          |
|  (转换为目标代码)      |
+-----------+------------+
            |
            v
+------------------------+
|    hello.o (目标文件)        |
+-----------+------------+
            |
            v
+------------------------+
|    链接 (Linking)          |
|  (生成可执行文件)      |
+-----------+------------+
            |
            v
+------------------------+
|    hello (可执行文件)        |
+------------------------+

以最简单的代码为例：

#include <stdio.h>

int main(void) {
    printf("Hello, Linux System Programming, World!\n");
    return 0;
}

预处理

通过gcc -E进行C文件的预处理：

$ gcc -E hello.c -o hello.i

预处理器处理代码中的预处理指令（以 # 开头），如宏定义、文件包含等。预处理的输出是一个扩展的源代码文件。

这里预处理主要做了以下操作：

文件包含：

  • #include <stdio.h> 指令被展开，stdio.h 文件的内容被插入到 hello.c 中。这包括标准输入输出函数和变量的声明。

宏定义和替换：

  • #define GREETING "Hello, World!\n" 指令定义了一个宏 GREETING，在后续代码中遇到 GREETING 时，都会被替换为 "Hello, World!\n"。

条件编译：

  • 如果代码中有条件编译指令（如 #ifdef 等），预处理器会根据条件的真假决定是否包含这些代码。

行号和文件名指令：

  • 预处理器还会添加一些指令来保持调试信息中的行号和文件名一致。

编译

使用gcc -S命令对.i文件进行编译，编译器将预处理后的代码转换为汇编代码，并生成汇编代码文件。

$ gcc -S hello.i -o hello.s

得到的汇编代码如下：

    .file	"hello.c"
    .text
    .section	.rodata
    .align 8
.LC0:
    .string	"Hello, Linux System Programming, World!"
    .text
    .globl	main
    .type	main, @function
main:
.LFB0:
    .cfi_startproc
    pushq	%rbp
    .cfi_def_cfa_offset 16
    .cfi_offset 6, -16
    movq	%rsp, %rbp
    .cfi_def_cfa_register 6
    movl	$.LC0, %edi
    call	puts
    movl	$0, %eax
    popq	%rbp
    .cfi_def_cfa 7, 8
    ret
    .cfi_endproc
.LFE0:
    .size	main, .-main
    .ident	"GCC: (GNU) 11.4.1 20230605 (Red Hat 11.4.1-2)"
    .section	.note.GNU-stack,"",@progbits

这部分就是从C语言得到的汇编代码，目的是从高级语言转为低级语言。

汇编

汇编器将汇编代码转换为目标代码（机器码）。这个步骤的输出是一个目标文件。

使用gcc -c命令将汇编语言转为目标代码（机器码）。

$ gcc -c hello.s -o hello.o
