背景

本文主要记录如何通过Cython调用C语言的库，之前了解过通过ctypes可以调用C语言的库，这次主要通过阅读官方文档，学习Cython如何调用C语言的库。

fprintf

example.pyx

# example.pyx

# 导入所需的C库函数
from libc.stdio cimport fprintf, stderr

# 定义外部函数
cdef extern from "stdio.h":
    int fprintf(FILE *stream, const char *format, ...)

# 调用C语言库函数
def call_fprintf():
    fprintf(stderr, "Hello, world from C with fprintf!\n")

setup.py

# setup.py

from setuptools import setup
from Cython.Build import cythonize

setup(
    ext_modules = cythonize("example.pyx")
)

编译命令：

$ python setup.py build_ext --inplace

运行结果：

$ python
Python 3.9.18 (main, Jan 24 2024, 00:00:00)
[GCC 11.4.1 20231218 (Red Hat 11.4.1-3)] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> import example
>>> example.call_fprintf()
Hello, world from C with fprintf!
>>>

这里主要关注这一行：

from libc.stdio cimport fprintf, stderr

这里会cimport一些C语言的方法，那么问题来了，我如何知道都有什么方法？以及方法是如何使用的？

fprintf和stderr是常用的C方法，那么有没有一个列表，可以查看都有哪些方法？
