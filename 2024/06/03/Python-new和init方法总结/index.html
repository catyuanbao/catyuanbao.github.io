背景

本文主要总结下Python __new__和__init__方法的区别与联系，以及和C++语言的new，构造方法的类比，掌握Python语言的对象初始化相关知识。

__new__方法用途

__new__ 是Python类中最容易滥用的功能之一。 它晦涩难懂，到处都是陷阱，当您确实需要 __new__ 时，它的功能就非常强大且难以理解。__new__ 的主要用例在元类中。

在Python以及许多其他语言中，对象分为两个步骤：

  1. 生成对象要访问对象,必须先创建该对象。对象创建就需要为其分配对应类型尺寸的内存空间，该对象存在的唯一标识符就是其内存实体的内存地址,由于Python中的所有对象都继承基类object，因此创建对象会调用object.__new__，而且__ new__会返回类的一个实例对象(通常指定为self)
  2. 构造对象：生成对象后，Python将该对象实例(通常用self表示,对于C底层就是该Py结构体的指针)传递给该类的构造函数 __init __,这一步通常是对类属性初始化或调用类内部的类方法

这个第一步，有点类似于C/C++的malloc和new，用于申请内存，后一步类似于C/C++的指针数据初始化以及构造方法。

__new__方法必须返回一个实例。

重写__new__方法

Python允许通过__new__ magic方法覆盖任何对象的__ new__。

class MyClass:
    def __new__(cls, *args, **kwargs):
        print("Creating instance")
        instance = super().__new__(cls)
        return instance

最经典的用途就是实现单例模式：

class Singleton:
    _instance = None

    def __new__(cls, *args, **kwargs):
        if cls._instance is None:
            cls._instance = super(Singleton, cls).__new__(cls)
        return cls._instance

    def __init__(self, value):
        self.value = value

# 创建实例
singleton1 = Singleton(10)
singleton2 = Singleton(20)

# 验证是否是同一个实例
print(singleton1 is singleton2)  # 输出: True

# 验证实例属性是否相同
print(singleton1.value)  # 输出: 20
print(singleton2.value)  # 输出: 20

这里主要是 cls._instance = super(Singleton, cls).__new__(cls)这一句，参数问题。

Singleton和cls。

  1. 在单继承情况下，super(Singleton, cls).__new__(cls) 等效于 object.__new__(cls)，但使用 super() 更加灵活，因为它不会硬编码父类名，这样如果将来修改继承关系，代码不需要改变。
  2. 在多重继承情况下，super() 能够确保按照 MRO 调用下一个类的方法，这样可以正确处理多重继承中复杂的继承关系。

多继承实例：

class Base1:
    def __new__(cls, *args, **kwargs):
        print("Base1 __new__ called")
        return super(Base1, cls).__new__(cls)

class Base2:
    def __new__(cls, *args, **kwargs):
        print("Base2 __new__ called")
        return super(Base2, cls).__new__(cls)

class Singleton(Base1, Base2):
    _instance = None

    def __new__(cls, *args, **kwargs):
        if cls._instance is None:
            cls._instance = super(Singleton, cls).__new__(cls)
        return cls._instance

    def __init__(self, value):
        if not hasattr(self, 'initialized'):
            self.value = value
            self.initialized = True

singleton1 = Singleton(10)
singleton2 = Singleton(20)

print(singleton1 is singleton2)  # 输出: True
print(singleton1.value)  # 输出: 10
print(singleton2.value)  # 输出: 10

一句话：super关键字是告知当前子类它指向子类“相对”的父类。

在__new__方法中使用属性初始化

下面的实例，没有__init__方法，只有__new__方法，在其中做属性的赋值，这种情况Python会自己加一个空的__init__方法。

class Person(object):
    def __new__(cls, name):
        instance = super(Person, cls).__new__(cls)
        instance.name = name
        return instance

tom = Person('tom')
print(tom.name)

显示调用__new__方法：

class Person(object):
    def __new__(cls, name):
        instance = super(Person, cls).__new__(cls)
        instance.name = name
        return instance

tom = Person('tom')
print(tom.name)

jerry = Person.__new__(Person, "jerry")
print(jerry.name)

自定义一个和空的__init__方法冲突的__init__方法，会报错：

class Person(object):
    def __new__(cls, name):
        instance = super(Person, cls).__new__(cls)
        instance.name = name
        return instance

    def __init__(self):
        print('call init')

tom = Person('tom')
print(tom.name)

jerry = Person.__new__(Person, "jerry")
print(jerry.name)

除非参数名称和默认的一致：

class Person(object):
    def __new__(cls, name):
        instance = super(Person, cls).__new__(cls)
        instance.name = name
        return instance

    def __init__(self, name):
        print('call init')

tom = Person('tom')
print(tom.name)

jerry = Person.__new__(Person, "jerry")
print(jerry.name)

输出：

call init
tom
jerry

说明，显式的调用__new__只会执行__new__，并不执行__init__方法。

对比C/C++

Python实例化对象方法：

class MyClass:
    def __new__(cls, *args, **kwargs):
        print("Creating instance")
        instance = super(MyClass, cls).__new__(cls)
        return instance

    def __init__(self, value):
        print("Initializing instance")
        self.value = value

obj = MyClass(10)

C++实例化对象方法：

#include 

class MyClass {
public:
    MyClass(int value) {
        std::cout << "Initializing instance" << std::endl;
        this->value = value;
    }

    static void* operator new(size_t size) {
        std::cout << "Creating instance" << std::endl;
        void* p = ::operator new(size);
        return p;
    }

private:
    int value;
};

int main() {
    MyClass* obj = new MyClass(10);
    return 0;
}
