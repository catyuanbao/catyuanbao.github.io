背景

本文主要记录解决AOC-2023-day3的问题过程，主要使用Python和Cython分别测试效果。

题目大意

第一部分

就精灵所能计算的而言，你必须计算出你在中奖号码列表中出现的号码。第一场比赛使这张牌值一分，第一场比赛后的每一场比赛使该牌的积分翻倍。

例如
卡片1:41 48 83 86 17 | 83 86 6 31 17 9 48 53
卡片2:13 32 20 16 61 | 61 30 68 82 17 32 24 19
卡片3:1 21 53 59 44 | 69 82 63 72 16 21 14 1
卡片4:41 92 73 84 69 | 59 84 76 51 58 5 54 83
卡片5:87 83 26 28 32 | 88 30 70 12 93 22 82 36
卡片6:31 18 13 56 72 |74 77 10 23 35 67 36 11

在上面的例子中，卡1有五个中奖号码（41、48、83、86和17）和八个号码（83、86、6、31、17、9、48和53）。

在你拥有的数字中，有四个（48、83、17和86）是中奖数字！

这意味着卡1值8分（第一场比赛得1分，第一场比赛后三场比赛各得三分）。
卡2有两个中奖号码（32和61），所以它值2分。
卡3有两个中奖号码（1和21），所以它值2分。
第4张卡有一个中奖号码（84），所以它值1分。
第5张牌没有中奖号码，因此不值得积分。
第6张卡没有中奖号码，因此不值得积分。
所以，在这个例子中，精灵的一堆刮刮卡值13分。
在一大堆五颜六色的卡片中坐下。它们总共值多少分？

这个问题就是有一个简单的查找问题，找到第二个数组中数字在第一个数组中出现的次数问题。

第二部分

就在你准备向精灵报告你的发现时，你们中的一个人意识到这些规则实际上一直印在每张卡片的背面。

没有所谓的“点”。相反，刮刮卡只会让你赢得更多的刮刮卡，等于你的中奖号码。

具体来说，您赢得的刮刮卡副本位于中奖卡下方，与比赛次数相等。所以，如果卡10有5个匹配的数字，你将赢得11、12、13、14和15张卡中的每一张。

刮刮卡的副本与普通刮刮卡一样记分，并且与复制的卡具有相同的卡号。

因此，如果你赢得了一张10号牌的副本，并且它有5个匹配的数字，那么它将赢得一张与原始10号牌赢得的相同的牌的副本：11、12、13、14和15号牌。

重复此过程，直到没有任何副本使您赢得更多的牌为止。（卡片永远不会让你把卡片复制到桌子的末尾。）

这一次，上面的例子有所不同：
卡片1:41 48 83 86 17 | 83 86 6 31 17 9 48 53
卡片2:13 32 20 16 61 | 61 30 68 82 17 32 24 19
卡片3:1 21 53 59 44 | 69 82 63 72 16 21 14 1
卡片4:41 92 73 84 69 | 59 84 76 51 58 5 54 83
卡片5:87 83 26 28 32 | 88 30 70 12 93 22 82 36
卡片6:31 18 13 56 72 |74 77 10 23 35 67 36 11

第1张牌有四个匹配的数字，因此您可以在接下来的四张牌中各赢一张：第2、3、4和5张牌。

你原来的第2张牌有两个匹配的号码，所以你赢得了第3张和第4张牌各一份。

你的第2张牌副本也会赢得第3张和第4张牌的副本。

你的四张牌3（一张正本和三张副本）有两个匹配的号码，所以你赢得了四张牌4和5。

你的八张牌4（一张正本和七张副本）有一个匹配的号码，所以你赢得了八张牌5的副本。

你的14张牌5（1张正本和13张副本）没有匹配的号码，也没有赢得更多的牌。

你的一张牌6（一张原件）没有匹配的号码，也没有赢得更多的牌。

处理完所有原件和副本后，您最终会得到1个卡实例1、2个卡实例2、4个卡实例3、8个卡实例4、14个卡实例5和1个卡示例6。总共，这个例子中的一堆刮刮卡会导致你最终拥有30张刮刮卡！

处理所有原始和复制的刮刮卡，直到不再赢得刮刮卡为止。包括原始的刮刮卡，你最终总共有多少张刮刮卡？

主要思路是使用一个队列，对得到的数据进行处理。

Python版本解决

import time
def read_data():
    data = []
    with open('in.txt') as fd:
        text_data = [line.strip() for line in fd]
        for line in text_data:
            nums, your_nums = line.split('|')
            data.append((list(map(int, (i for i in nums.split(':')[1].strip().split(' ') if i))),
                list(map(int,
                (i for i in your_nums.split(' ') if i)))))
    return data


def compute_sum(line_data):
    total = 0
    for line in line_data:
        your_score_times = 0
        nums, your_nums = line[0], line[1]
        for i in your_nums:
            if i in nums:
                your_score_times += 1
        if your_score_times == 0:
            total += 0
        else:
            total += pow(2, your_score_times-1)
    return total

data = read_data()
s = time.time()
for i in range(10000):
    compute_sum(data)
print(time.time()-s)

13.626471519470215

from collections import defaultdict, deque
import time

def read_data():
    data = []
    with open('in.txt') as fd:
        text_data = [line.strip() for line in fd]
        for line in text_data:
            nums, your_nums = line.split('|')
            data.append((set(map(int, (i for i in nums.split(':')[1].strip().split(' ') if i))),
                         set(map(int, (i for i in your_nums.strip().split(' ') if i)))))
    return data

def compute_same_count(line):
    return len(line[0] & line[1])

def compute_sum(line_data):
    cache = {}  # 缓存，用来记录每行相同数字的个数
    queue = deque()
    total = 0
    card_instances = defaultdict(int)

    # 初始化队列
    for index, line in enumerate(line_data):
        same_count = compute_same_count(line)
        cache[index] = same_count
        card_instances[index] += 1
        if same_count > 0:
            for i in range(1, same_count + 1):
                new_card_index = index + i
                if new_card_index < len(line_data):
                    queue.append(new_card_index)
                    card_instances[new_card_index] += 1

    # 处理队列
    while queue:
        current_card = queue.popleft()
        if current_card in cache:
            same_count = cache[current_card]
        else:
            same_count = compute_same_count(line_data[current_card])
            cache[current_card] = same_count

        if same_count > 0:
            for i in range(1, same_count + 1):
                new_card_index = current_card + i
                if new_card_index < len(line_data):
                    queue.append(new_card_index)
                    card_instances[new_card_index] += 1

    # 计算拥有的卡片数量
    total = sum(card_instances.values())
    return total

data = read_data()
s = time.time()
for i in range(10):
    res = compute_sum(data)
print(time.time()-s)
