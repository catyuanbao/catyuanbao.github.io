递归深度

测试各类编程语言的递归最大深度。C语言居然可以从main方法调用main方法。

int count = 0;

main() {
    printf("call main %d times\n", count);
    count++;
    main();
}

Python语言默认最大只有1000.

C语言没有内置的最大递归深度限制，这取决于具体的系统和编译器设置。递归深度主要是由堆栈大小决定的。

使用ulimit设置更大的堆栈大小，就能增加递归深度。

Python语言可以通过setrecursionlimit方法重新设置递归深度：

import sys

# 查看当前递归最大深度
print(f"当前递归最大深度: {sys.getrecursionlimit()}")

# 设置新的递归最大深度
sys.setrecursionlimit(2000)
print(f"新的递归最大深度: {sys.getrecursionlimit()}")

# 定义一个递归函数
def recursive_function(n):
    if n == 0:
        return 0
    else:
        return 1 + recursive_function(n - 1)

try:
    print(recursive_function(1500))
except RecursionError as e:
    print(f"递归错误: {e}")

递归阶乘问题

C++使用递归求解阶乘问题：

#include 

using namespace std;

int factorial(int);

int main() {
    cout << "Please enter a non-negative int: ";
    int n;
    cin >> n;

    cout << "Factorial if " << n << " is " << factorial(n) << " \n";

    return 0;
}

int factorial(int n) {
    return n == 0 ? 1 : n * factorial(n - 1);
}

这里输入整数过大会造成溢出。

递归计算斐波那契数列问题

使用递归解决这个问题效率比较低，最好加上缓存。

int fib(int index) {
    if (index == 0) {
        return 0;
    } else if (index == 1) {
        return 1;
    } else {
        return fib(index - 1) + fib(index - 2);
    }
}

递归解决回文判断

def is_palindrome(s):
    length = len(s)
    if length <= 1:
        return True
    elif s[0] != s[length - 1]:
        return False
    else:
          # 去掉头尾字符继续比较
        return is_palindrome(s[1: length - 1])


print(is_palindrome("aba"))
print(is_palindrome("abac"))

这个方法，性能比较差，可以使用一个辅助函数优化性能，类似双指针，这样做的好处是不会产生更多的新字符串的副本：

def helper_palindrome(s: str, low: int, high: int):
    if high <= low:
        return True
    elif s[low] != s[high]:
        return False
    else:
        return helper_palindrome(s, low + 1, high - 1)



def is_palindrome(s):
    return helper_palindrome(s, 0, len(s) - 1)

print(is_palindrome("aba"))
print(is_palindrome("abac"))

递归返回数组中的最大元素值

from typing import List

def max_value(nums: List[int]) -> int:
    """递归获取最大值"""
    length = len(nums)
    if length == 1:
        return nums[0]
    else:
        max_value_other = max_value(nums[1: length])
        return nums[0] if nums[0] > max_value_other else max_value_other


print(max_value([1,2,3,4,5,6]))
print(max_value([10,2,3,4,5,6]))

递归逆序输出数字

递归逆序输出每位数字：

def reverse_print(n):
    # 处理负数的情况
    if n < 0:
        print('-', end='')
        n = -n

    # 基本情况：n 是个位数
    if n < 10:
        print(n, end='')
    else:
        # 打印最后一位数字
        print(n % 10, end='')
        # 递归调用去掉最后一位的数字
        reverse_print(n // 10)

递归逆序打印字符串

def reverse_print_str(s):
    if len(s) == 1:
        print(s[0], end='')
    else:
        print(s[-1], end='')
        # 去掉最后一位递归调用
        reverse_print_str(s[:len(s) - 1])

递归计算整数中所有数字之和

def compute_all_sum(num):
    if num < 0:
        num = -num

    if num < 10:
        return num

    single_num = num % 10
    other_num = compute_all_sum(num // 10)
    return single_num + other_num

print(compute_all_sum(12345))
print(compute_all_sum(123456))

递归统计一个字符在指定字符串中出现的次数

def occor_times(s, c):
    if not s:
        # 空字符串
        return 0
    elif s[0] == c:
          # 相等，+1 处理后续字符串
        return 1 + occor_times(s[1:], c)
    else:
        return occor_times(s[1:], c)

print(occor_times('Welcome', 'e'))

打印一个字符串所有排列

def permute(s, answer):
    if len(s) == 0:
        print(answer)
        return

    length = len(s)
    for i in range(length):
        ch = s[i]
        left_substr = s[:i]
        right_substr = s[i+1:]
        rest = left_substr + right_substr
        # rest是剩余部分，answer是当前已经排列好的部分。
        permute(rest, answer + ch)

string = "abcde"
permute(string, "")

递归实现数组中所有正数的和

def sum_of_nums(nums):
    """递归实现返回所有正数的和"""
    if not nums:
        return 0
    if nums[0] > 0:
        return nums[0] + sum_of_nums(nums[1:])
    else:
        return sum_of_nums(nums[1:])

print(sum_of_nums([1,2,3,4,-1,-2,-3,0]))
